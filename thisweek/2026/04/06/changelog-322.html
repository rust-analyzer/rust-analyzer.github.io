Universal VSIX

This release adds a platform-independent VSIX that VS Code will install on platforms where rust-analyzer doesn’t offer binaries. On supported platforms, the extension should keep working as before, but a Code bug might cause it to fail to activate. If that happens, you should uninstall and reinstall the rust-analyzer extension.
