Announcement

This release brings two major changes, which might make it slightly spicier than usual.

The first is upgrading to the latest version of salsa, the incremental evaluation framework used by rust-analyzer. This will make it possible to implement parallel evaluation and persistency in the future.

The second change is to use salsa for the crate graph, making it incremental. Adding or removing a dependency, or editing a build script or proc macro, will now invalidate only the affected crates instead of the entire workspace.
